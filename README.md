# Научная визуализация скалярных полей

Данный репозиторий содержит программный код для курсовой работы по дисциплине «Проектная деятельность в программировании и научных вычислениях». В работе рассматриваются методы визуализации скалярных полей на примере электрического потенциала системы точечных зарядов.

## Цель работы

Разработать программную реализацию визуализации электрического потенциала системы точечных зарядов и провести сравнительный анализ методов визуализации по критериям наглядности, информативности и скорости.

## Реализованные методы визуализации

- **Тепловая карта (heatmap)** — цветовое отображение распределения потенциала.
- **3D-поверхность** — трёхмерное представление поля.
- **Изолинии (собственная реализация Marching Squares)** — извлечение изолиний с использованием алгоритма побитового кодирования и линейной интерполяции.
- **Изолинии (встроенная функция find_contours)** — эталонный метод для сравнения.

## Используемые технологии

- Python 3.8+
- NumPy — для численных расчётов
- Matplotlib — для построения графиков
- scikit-image — для встроенной функции извлечения контуров
